# Sistema de Gestión de Colas - Laboratorio Clínico

Sistema de gestión de colas para laboratorio clínico desarrollado con FastAPI y Angular.

## Características

- ✅ Gestión de pacientes y tickets
- ✅ Control de ventanillas/estaciones
- ✅ Sistema de notificaciones SMS
- ✅ Pantallas en tiempo real
- ✅ Reportes y analytics
- ✅ Panel administrativo

## Tecnologías

- **Backend**: Python FastAPI
- **Base de Datos**: SQL Server
- **Cache**: Redis
- **Frontend**: Angular (próximamente)

## Instalación

1. Clonar el repositorio
2. Instalar dependencias: `pip install -r requirements.txt`
3. Configurar variables de entorno: `.env`
4. Ejecutar migraciones: `alembic upgrade head`
5. Iniciar servidor: `uvicorn app.main:app --reload`
